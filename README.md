Document Parser

Simple Spring Boot demo that accepts file uploads and parses documents (PDF).

**Quick Start:**
- **Build:** `./mvnw package` (or `mvn package`)
- **Run:** `./mvnw spring-boot:run` (or `mvn spring-boot:run`)
- **Open:** `http://localhost:8080/` to access the sample `index.html` upload page.

**Requirements:**
- **Java:** 17
- **Build tool:** Maven (wrapper included)

**Notes:**
- The project uses Apache PDFBox for PDF processing (see `pom.xml`).
- Main source is in `src/main/java/com/example/demo` and static content in `src/main/resources/static`.
- To push changes to GitHub, the remote `origin` is already set to the repository used for this workspace.
